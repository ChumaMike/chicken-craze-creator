# 🍗 Chicken Craze

Online ordering and presence site for a local chicken takeaway — menu, ordering flow, and delivery zones.

🔗 Live: _(link once deployed)_

---

## Stack

| Layer | Tech |
| :--- | :--- |
| Framework | Vite · React · TypeScript |
| Edge runtime | Cloudflare Workers |
| Styling | Tailwind · shadcn/ui · Radix primitives |
| Testing | Playwright |
| Package manager | Bun |

---

## Local development

```bash
bun install
bun run dev
```

### Deploy to Cloudflare

```bash
bun run build
wrangler deploy
```
